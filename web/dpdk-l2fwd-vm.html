Experiment

This command uses four experiment nodes of a testbed, the loadgen-node, the dut-node, the timestamper-node, and the evaluator-node. The loadgen, dut and timestamper nodes must be wired according to the setup description.

Executed command 

# rt kernel
./experiment.sh loadgen-node dut-node timestamper-node evaluator-node vm dpdk-l2fwd-cat optimized bullseye-rt 60
# vanilla kernel
./experiment.sh loadgen-node dut-node timestamper-node evaluator-node vm dpdk-l2fwd-cat optimized bullseye-default 60
# nohz kernel
./experiment.sh loadgen-node dut-node timestamper-node evaluator-node vm dpdk-l2fwd-cat optimized bullseye-nohz 60

The experiment.sh file is used to initiate the experiment workflow. The run.sh file is called by the experiment.sh file.
