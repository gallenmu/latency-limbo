Reproducing Evaluation Results

This page describes the scripts and data necessary to reproduce the evaluation for the paper: "How Low Can You Go? Latency Limbo for Intrusion Prevention Systems".

pos testbed

The experiment on this page were created using the pos framework. pos is a experiment framework and testbed that automates the experiment workflow to create reproducible network experiments. pos was presented at CoNEXT'21.

Sebastian Gallenmüller, Dominik Scholz, Henning Stubbe, Georg Carle, The pos Framework: A Methodology and Toolchain for Reproducible Network Experiments, in The 17th International Conference on emerging Networking EXperiments and Technologies (CoNEXT ’21), Munich, Germany (Virtual Event), Dec. 2021.
